dlsHome Snapclient Installer

Flash the maintained ESP32-S3 firmware for the dlsHome PCM5102A board.

This installer supports only the two maintained builds from this fork: Wi-Fi and Ethernet.

Use the Wi-Fi image for the standard wireless setup. Use the Ethernet image only for the W5500-based wired configuration.

Not Allowed
Flashing is only supported from localhost or from a secure website.
Browser Not Supported Make sure you are using a compatible browser

Maintained firmware for the dlsHome PCM5102A board on ESP32-S3 with 16 MB flash and PSRAM.

Use this image when the device will connect over Wi-Fi. After flashing, configure your local SSID and password through the firmware's supported setup flow.

Network: Wi-Fi

Target: ESP32-S3 + PCM5102A + PSRAM

Maintained wired firmware for the same dlsHome ESP32-S3 hardware using the W5500 SPI Ethernet configuration from this repository.

Use this image only if your board is wired for the supported Ethernet overlay. It replaces Wi-Fi at build time; the firmware does not run both at once.

Network: Ethernet via W5500

Target: ESP32-S3 + PCM5102A + PSRAM

Only the Wi-Fi and Ethernet dlsHome builds are supported in this fork. Legacy boards and older installer targets have been removed from this UI.
